Pancake Cooking Timer

Start the timer to cook a perfect pancake (90 seconds per side)!

90

Code Used:

        <script>
        let timeLeft = 90;
        let timer;

        function startTimer() {
            document.getElementById("timer").textContent = timeLeft;
            timer = setInterval(() => {
                timeLeft--;
                document.getElementById("timer").textContent = timeLeft;
                if (timeLeft === 0) {
                    clearInterval(timer);
                    alert("Flip the pancake!");
                }
            }, 1000);
        }
        </script>
